Web Design?

Published on January 27, 2024

Hello again! So, today I wanted to write a little about why I decided to make a portfolio website and give some insight into how the process went for me.

First and foremost, it's worth mentioning that before setting this up I had no experience with web design (aside from basic HTML) or even using Git for versioning; creating this website has been immensely beneficial in both of those regards. Starting this project, I had heard of GitHub pages and was vaguely aware that it let you host a website with a cool address so the first step I took, as any good researcher will do, was search on Google 'How to make a website with GitHub.' Digging deeper I learned what repositories are, how versioning works, and questioned why I hadn't begun using Git earlier.

Once I had a gist of what I was doing, I found a nice template (courtesy of html5up.net) and began making gradual edits and committing them to Git frequently. I relied on both ChatGPT and Google Bard for help with coding, particularly with CSS as I am entirely unfamiliar with this language, and changed up the template significantly. Eventually, we ended up with the current iteration! That isn't too say there won't be more changes coming soon, as I do have a few ideas for how to improve thanks to some feedback I received. I'll keep you updated.
